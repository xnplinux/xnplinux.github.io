XNP Linux Installation Guide

Follow these steps to deploy the meta-distribution inside your host environment.

Step 1: Download & Extract Rootfs

Download the latest release and extract it to your desired chroot directory:

wget https://github.com/xnplinux/xnplinux.github.io/releases/download/v1.2.0/xnplinux-v1.2.0.tar.gz or download tar.gz from releases
mkdir rootfs 
sudo tar -xvf xnplinux-v1.2.0.tar.gz -C rootfs
cd rootfs

Step 2: Enter Environment & Build Packages

Mount essential filesystems, chroot into the system, and compile your software:

sudo mount -o bind /dev /home/your_name/rootfs/dev
sudo chroot rootfs /bin/sh -l
mount -t proc proc ./proc
xnp -b package_name
xnp -i package_name

Step 3: Verify and Run

Launch your newly compiled binary to verify the installation. If the application fails to start, ensure the virtual filesystems are mounted properly:

package_name
