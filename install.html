XNP Linux Installation Guide

Follow these steps to deploy the meta-distribution inside your environment.

Step 1: Download xnp linux

In your system, type this:

wget https://github.com/xnplinux/xnplinux.github.io/releases/tag/v1.0.0/xnplinux-v1.0.tar.gz


Step 2: Building the System


Compile and install core utilities using the xnp packager.


xnp -b  package
xnp -i  package




Step 3: Verifying Installation


Run the dynamic info utility to verify the deployment:


package_name
if not run,try this mount -t proc proc /proc, and try again
