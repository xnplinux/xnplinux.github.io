XNP Linux Installation Guide

Follow these steps to deploy the meta-distribution inside your environment.

Step 1: Preparing Host & Directory

Create an isolated playground on your host system and move inside:

mkdir -p /mnt/
cd /mnt/

Step 2: Download xnp linux

In your system, type this:

wget https://github.com/xnp/is/soon/sorry
tar xvf xnp-chroot-26.tar.xz
Jump in your system! (don't forget exit from /mnt)
/mnt/xnp/xnp-chroot /mnt

Step 3: Repository Setup

Clone the official core repository and configuration templates:

mkdir -p root/repos && cd root/repos
git clone https://sorry.xnp.is.soon!

Step 4: Environment Configuration

Lock the XNP system paths permanently into your profile setup:

echo 'export XNP_PATH="/root/repos/repo/core:$XNP_PATH"' >> ~/.profile
source ~/.profile

Step 5: Building the System

Compile and install core utilities using the xnp packager.

xnp -b  package
xnp -i  package

Step 6: Verifying Installation

Run the dynamic info utility to verify the deployment:

package_name
