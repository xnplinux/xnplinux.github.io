# XNP Linux

A POSIX-compliant, lightweight meta-distribution built for absolute control. 

Designed to run seamlessly inside any host environment via chroot hosting. Inspired by the KISS Linux philosophy.

## Features
* **POSIX sh:** The entire package manager (`xnp`) is written in lightweight, standard shell script.
* **Independent & Lightweight:** Control every single byte of your system.

## Quick Start
To install XNP Linux, follow the official installation guide on our website:
